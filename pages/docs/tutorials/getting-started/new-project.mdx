# Create a New Effect Project

In this tutorial, you'll learn how to get started with Effect from scratch using a plain **TypeScript** project.

<Steps>
### Create a project directory and navigate into it

```sh
mkdir hello-effect
cd hello-effect
```

### Initialize a TypeScript project using npm

```sh
npm init -y
npm install typescript ts-node @types/node --save-dev
```

This creates a `package.json` with an initial setup for your TypeScript app.

### Initialize TypeScript

```sh
npx tsc --init
```

### Install the necessary packages

```sh
npm install @effect/io @effect/data
```

These packages will provide the foundational functionality for your Effect project.

</Steps>
